Healthcare Regional Staffing Database

Overview
This project is the design and implementation of a relational SQL database to model a healthcare staffing system. The database tracks regions, directors, teams, nurses, and contact information.

Key Features
- Normalized relational database structure
- Primary and foreign key relationships
- Multi-table JOIN queries
- Aggregation and ranking analysis using SQL

Technologies
- SQL Server
- T-SQL

How to Run
1. Run `01-schema.sql` to create the database and tables
2. Run `02-data.sql` to insert sample data
3. Run `03-queries.sql` to execute analysis queries

Example Analysis
- Nurse distribution by region
- Staffing by director
- Team workload comparison
- Organizational hierarchy reporting
- Ranking teams by size

Files
- 01-schema.sql
- 02-data.sql
- 03-queries.sql

Note
This project uses self-created sample data for demonstration and portfolio purposes.
